# html

## html 이란?

웹페이지를 기술하기 위한 마크업 언어

여기서 마크업 언어는 문서, 데이터의 구조를 명시 한 언어

## 기본 문법

기본적으로 한 태그에 시작과 끝이 있음

예외의 경우는 빈 태그라고 부름 가령

```html
<br />
<img />
```

같은 경우

## 시멘틱 웹

```html
<font size="6"><b>Hello</b></font>
<h1>Hello</h1>
```

위의 두 hello 는 브라우저 화면으로 보기에는 둘다 같은 크키의 hello 임

하지만 무슨 의미인지는 알 수 없음

뭔소리냐면 두번째 줄의 h1 태그를 보고 두번째 줄은 제목이 hello 라는 걸 알 수 있음

이게 시멘틱 태그임

시맨틱 태그란 브라우저, 검색엔진, 개발자 모두에게 콘텐츠의 의미를 명확히 설명하는 역할

    시맨틱 웹이란 웹에 존재하는 수많은 웹페이지들에 메타데이터(Metadata)를 부여하여,

    기존의 잡다한 데이터 집합이었던 웹페이지를 ‘의미’와 ‘관련성’을 가지는 거대한 데이터베이스로 구축하고자 하는 발상

그리고 시멘틱 태그가 있고 non-시멘틱 태그가 있음

시멘틱 코드가 아닌 태그는 div, span 같은 애들을 말함
